== Description ==

Use this plugin to set multiple contributors of single post, And show them after post content. Simply select those from post editor window. After plugin installation post editor show 'MG Contributors' window. At that window select multiple authors of this post and publish it. Make sure 'Enable List' option is selected. If 'Enable List' checkbox unchecked then it does not show contributors list.

Development for this plugin can be found on 
GitHub: [https://github.com/maheshwaghmare/mg-post-contributors]

== Installation ==

1. Download and Upload `mg-post-contributors` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins'
3. Select New post i.e. "Post->Add Post" or Select existing post "Post->All Posts".
4. Select Authors from 'MG Contributors' window from post editor window.
5. Click on Publish / Update.
6. To see result click on "View Post".


== How to Use ==
You can set contributors of the post for NEW or EXISTING Post. Follow below steps as per your requirement.

<strong> 1. New Post: </strong> 
	<ul>
		<li>Create new post by "POST->NEW POST". </li>
		<li>Enter post Title, Description and click "PUBLISH". </li>
		<li>Add Contributors / Authors. </li>
		<li>To see result click "View Post". </li>
	</ul>

<strong> 2. Existing Post: </strong> 
	<ul>
		<li>Click "POST->ALL POST", </li>
		<li>Select Contributors / Authors.</li>
		<li>Click on "UPDATE". </li>
		<li>To see result click "View Post". </li>
	</ul>
